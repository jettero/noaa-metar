About NOAA METAR

About

Written by Paul Miller.

This is the “Lite” version of this app. There is a heavier version that attempts to decode the METAR and TAF data as a learning tool. If you find you have no idea how to read the data prestended by this app, you should probably try NOAA METAR instead. Please note that the decoder and airport database are very US-centric and as such, the standard version is not available internationally.

This app is all about METAR. Attempts to process the METAR data are purely for educational and learning purposes. This app is intended only for people who wish to learn to read METAR or already know how to do so.

This is the standard version of this app. There is also a much lighter version that does not have the airport database nor the METAR decoder. The app is considerably smaller without these things and therefore loads quite a bit faster. If you already know how to read METAR and TAF, you should probably give METAR Lite a try.

You may report bugs and request features by using the github issue tracker:

http://github.com/jettero/noaa-metar/issues

If it is easier, you may also email support, bug, and feature requests to

noaa-metar-feedback@voltar.org

Problems with METAR decoding should be reported using the form built into the decode page. If you think there's a problem with a particular decode, select “Report Metar Decode” from the app menu on the Decode card.

The entire source for this app can be found on github:

http://github.com/jettero/noaa-metar

Some unreleased, release candidate and historical builds can be found on the gh-pages:

http://jettero.github.com/noaa-metar
