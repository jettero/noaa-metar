About NOAA METAR

About

Written by Paul Miller.

This app is all about METAR. Attempts to process the METAR data are purely for educational and learning purposes. This app is only intended for people who wish to learn to read METAR or already know how to do so.

You may report bugs and request features by using the github issue tracker:

http://github.com/jettero/noaa-metar/issues

If it is easier, you may also email support, bug, and feature requests to

noaa-metar-feedback@voltar.org

Problems with METAR decoding should be reported using the form built into the decode page. If you think there's a problem with a particular decode, select “Report Metar Decode” from the app menu on the Decode card.

The entire source for this app can be found on github:

http://github.com/jettero/noaa-metar

Some unreleased, release candidate and historical builds can be found on the gh-pages:

http://jettero.github.com/noaa-metar
