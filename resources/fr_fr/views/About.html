A propos

Écrit par Paul Miller.

Il s'agit de la version "Lite" de ce soft. Il ya une version plus lourde que les tentatives pour décoder les METAR et TAF de données comme outil d'apprentissage. Si vous trouvez que vous avez aucune idée de comment lire les données prestended par cette application, vous devriez probablement essayer NOAA METAR lieu. S'il vous plaît noter que la base de données décodeur et de l'aéroport sont très centrés sur les É.-U. et en tant que telle, la version standard n'est pas disponible au niveau international.

Ce soft est tous les METAR sujet. Les tentatives visant à traiter les données METAR sont purement à des fins éducatives et d'apprentissage. Cette application est destinée uniquement aux personnes qui souhaitent apprendre à lire METAR ou savent déjà comment faire.

Il s'agit de la version standard de ce soft. Il ya aussi une version beaucoup plus légère qui n'a pas la base de données de l'aéroport, ni le décodeur METAR. Le soft est nettement plus faible, sans ces choses et donc des charges un peu plus rapide. Si vous savez déjà comment lire METAR et TAF, vous devriez probablement donner METAR Lite essayer.

Vous pouvez rapporter les bugs et fonctionnalités demande en utilisant le suivi de problèmes github:

http://github.com/jettero/noaa-metar/issues

Si il est plus facile, vous pouvez aussi envoyer un courriel de soutien, bug, et demandes de fonctionnalités à

noaa-metar-feedback@voltar.org.

Problèmes avec le décodage METAR doivent être déclarés en utilisant le formulaire intégré dans la page de décoder. Si vous pensez qu'il ya un problème avec un décodeur spécifique, sélectionnez "Problème Decode Rapport" dans le menu application sur la carte de décodage.
