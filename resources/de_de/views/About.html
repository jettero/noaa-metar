Über

Geschrieben von Paul Miller.

Dies ist die "Lite-Version" dieses App. Es gibt eine schwerere Version, die versucht, die METAR und TAF-Daten als Lerninstrument zu dekodieren. Wenn Sie feststellen, haben Sie keine Ahnung, wie app lesen Sie die diese Daten durch prestended, sollten Sie vielleicht versuchen NOAA METAR statt. Bitte beachten Sie, dass der Decoder und Flughafen-Datenbank sehr US-zentrisch und als solche sind, ist die Standard-Version nicht international zur Verfügung.

Diese App ist alles über METAR. Versuche, den Prozess METAR Daten werden ausschließlich für Bildungs-und Lernzwecken. Diese Anwendung ist nur für Leute, die lernen, METAR lesen oder wissen bereits, wie Sie dies tun wollen.

Dies ist die Standard-Version dieser App. Es gibt auch eine viel leichtere Version, die nicht über den Flughafen Datenbank noch die METAR-Decoder. Die App ist wesentlich kleiner, ohne diese Dinge und deshalb lädt recht ein bisschen schneller. Wenn Sie bereits wissen, wie man lesen METAR und TAF, sollten Sie vielleicht geben METAR Lite versuchen.

Sie können Fehler und Gesuche für neue Funktionen mithilfe der github issue tracker Bericht:

http://github.com/jettero/noaa-metar/issues

Wenn es einfacher ist, können Sie auch Email-Support, Bug-und Feature-Requests zu:

noaa-metar-feedback@voltar.org.

Probleme mit METAR Dekodierung sollten anhand des Formulars in das decodieren Seite gebaut werden. Wenn Sie denken, es gibt ein Problem mit einem bestimmten entschlüsseln, wählen Sie "Bericht Decode Problem" aus dem App-Menü auf der Decode-Karte.

Der gesamte Quellcode für diese Anwendung kann auf github finden Sie hier:

http://github.com/jettero/noaa-metar

Einige unveröffentlichte, Release Candidate und historischen baut auf die gh-pages gefunden werden:

http://jettero.github.com/noaa-metar
